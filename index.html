👨‍💻 My DevOps Journey

I began my journey with curiosity about how code turns into running applications. Over time, I explored automation, containerization, cloud services, and CI/CD pipelines. Today, I thrive on bridging gaps between development and operations using tools like Jenkins, Docker, Kubernetes, Ansible, Terraform, AWS, Prometheus, and Grafana.

🔄 DevOps in the SDLC

DevOps integrates across every phase of the Software Development Life Cycle:

  • Plan: Agile tools for planning sprints and tasks
  • Develop: Git for version control, code collaboration
  • Build: Maven/Gradle to compile and package
  • Test: Automated testing using Jenkins pipeline
  • Release: Docker + Ansible for packaging and environment config
  • Deploy: Kubernetes for orchestrated deployment on EKS
  • Monitor: Prometheus & Grafana for metrics & visualization
